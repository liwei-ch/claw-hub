#### ▶️ 抄底操作核心风险管控（严格遵循）
🔴 5大核心风险点：
1. 趋势误判风险：即使有预判能力、敢于左侧布局，也容易低估趋势的力量，左侧抄底后可能面临趋势性下跌的长期压力。
2. 仓位管理风险：过度自信导致重仓、满仓抄底，过早打光全部资金，没有预留后手应对后续更深幅度的下跌，陷入被动。
3. 止损失效风险：抄底后股价继续下跌，不愿认错止损，原本的短期投机抄底操作演变成长期价值投资死扛，资金被长期套牢。
4. 认知盲区风险：即使精通基本面、估值、政策分析，也容易主观判断"已经跌到位"，忽略黑天鹅事件、流动性枯竭、监管政策突变、资金负反馈等极端风险因素。
5. 扛不住最后一公里风险：即使最终判断底部正确，也可能因为扛不住回撤、杠杆爆仓、心态崩溃、资金期限错配等问题，在行情反转前的最后阶段被迫平仓，倒在黎明前。

#### ▶️ 滚动交易（做T）规则
1. **核心逻辑**：保留固定底仓前提下，利用价格波动低吸高抛，全程不改变总持仓量，摊薄持仓成本
2. **两种操作模式**：
   - 正向滚动（先买后卖）：预判上涨时先买等量筹码，拉升后卖出旧仓，当日闭环，适用于上升趋势
   - 反向滚动（先卖后买）：预判下跌时先卖部分底仓，下跌后买回等量筹码，当日闭环，适用于震荡市
3. **铁则约束（严格执行）**：
   - 仅允许对中长期趋势向上的标的操作，下跌趋势标的绝对禁止
   - 单次滚动仓位不超过底仓的1/2，禁止满仓做T
   - 预期波动空间≥2%才操作，避免手续费摩擦损失
   - 当日必须完成买卖闭环，不得改变底仓数量
   - 操作错误亏损1%立即止损，禁止硬扛

### 5. 经验教训
1. **2026-03-16 板块统计错误教训**：统计板块涨停/异动情况时，禁止仅用少量样本股统计，必须覆盖≥20只板块核心标的，优先采用「全市场涨停个股按板块归类」的方式统计，统计完成后需交叉复核2次，确保数据准确，避免因样本不全导致判断错误。
2. **2026-03-16 板块热点判断规则完善**：板块异动/热点判断采用8维度综合判断标准：
   - 核心条件（必须同时满足）：平均涨幅≥2% + 涨停≥3只 + 上涨占比≥70%
   - 辅助条件：大阳（≥5%）≥10只 + 大阴（≥5%）≤3只 + 跌停≤1只 + 主力资金净流入≥5亿元
   - 按达标情况分为三级热点：超强/较强/普通
   - 涨停判断按板块差异化标准：主板≥9%、科创板/创业板≥18%、北交所≥27%
3. **2026-03-16 成交量判断错误教训**：判断放量/缩量不能仅看单日成交量绝对值，必须和近5日/10日平均量能对比，按量比判断：
   - 量比≥1.5 → 放量
   - 量比≤0.7 → 缩量
   - 0.7<量比<1.5 → 平量
   - 禁止仅用单日绝对值判断放量/缩量
