## 1. Baostock

### 概述
免费开源的A股历史数据接口，专注于提供高质量的A股历史行情和财务数据。

### 申请步骤
1. **无需注册申请**，完全免费开源
2. 直接通过pip安装即可使用：
```bash
pip install baostock
```

### 获取token/key流程
- **不需要token或key**
- 直接调用接口，无需身份认证

### 调用方式
```python
import baostock as bs
import pandas as pd

# 登录系统
lg = bs.login()

# 获取沪深A股历史K线数据
rs = bs.query_history_k_data_plus(
    "sh.600000",  # 股票代码，格式：交易所+股票代码
    "date,code,open,high,low,close,volume,amount",  # 指标
    start_date='2017-01-01',
    end_date='2017-12-31',
    frequency="d",  # d=日k线, w=周, m=月, 5=5分钟, 15=15分钟, 30=30分钟, 60=60分钟
    adjustflag="3"  # 1:后复权 2:前复权 3:不复权
)

# 转成DataFrame
data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
df = pd.DataFrame(data_list, columns=rs.fields)

# 登出系统
bs.logout()
```

### 注意事项
- 股票代码格式要求：`sh.600000`（上交所）、`sz.000001`（深交所）
- 服务器位于境外，网络不稳定时可能连接失败
- 数据更新延迟一般为T+1日
- 分钟线数据只提供最近几年的历史数据

### 频率限制
- 官方未明确公布具体频率限制
- 实际使用建议单次请求不超过1000条记录，避免被封禁IP

### 免费额度
- **完全免费**，无额度限制
- 支持获取：
  - 日线/周线/月线数据
  - 5/15/30/60分钟线数据
  - 财务数据、指数数据、板块数据
  - 资金流向、龙虎榜数据

## 2. Tushare / Tushare Pro

### 概述
国内最流行的金融数据接口，采用积分制，基础数据免费，高级数据需要积分。

### 申请步骤
1. 访问 [Tushare官网](https://tushare.pro/) 注册账号
2. 完成邮箱验证和个人信息填写
3. 登录后在「个人中心」->「接口TOKEN」获取token

### 获取token/key流程
1. 注册登录后，点击右上角头像 → `个人中心`
2. 在左侧菜单找到 `接口TOKEN`
3. 点击 `生成新TOKEN` 即可获得（可重新生成）
4. **token需要保密，不要泄露**

### 调用方式
1. 安装：
```bash
pip install tushare
```

2. 调用：
```python
import tushare as ts

# 设置token
pro = ts.pro_api('你的token在这里')

# 获取日线行情
df = pro.daily(ts_code='000001.SZ', start_date='20180701', end_date='20180718')

# 获取股票列表
stocks = pro.stock_basic(exchange='', list_status='L', fields='ts_code,symbol,name,area,industry,list_date')
```

> 注意：股票代码格式为 `000001.SZ`，后缀为 `.SH`(上交所)、`.SZ`(深交所)、`.BJ`(北交所)

### 注意事项
- 采用**积分制**，不同接口需要不同积分权限
- 新用户注册默认赠送 **100积分**
- 通过邀请好友、每日签到可以获得更多积分
- 数据质量较高，更新及时
- 社区活跃，文档齐全

### 频率限制
- 免费用户：**每分钟最多调用20次**，每天最多调用10000次
- 积分越高，频率限制越宽松

### 免费额度
| 积分范围 | 可用接口 | 说明 |
|---------|---------|------|
| 0-100 | 基础数据（股票列表、日线行情、交易日历等） | 新用户注册即有 |
| 100+ | 大部分常用接口（财务数据、指数、板块等） | 邀请1人获得20积分 |
| 500+ | 全部接口（包括分钟线、tick数据等） | 需要邀请或积分充值 |

- **充值获得积分**：1元 = 10积分，最低充值10元（100积分）
- 基础量化研究，100积分足够使用

## 3. AkShare

### 概述
开源免费的Python金融数据分析接口，集成了多个数据源，无需注册即可使用。

### 申请步骤
1. **无需注册申请**，完全免费开源
2. 直接通过pip安装：
```bash
pip install akshare
```

### 获取token/key流程
- **大部分接口不需要token或key**
- 少数第三方数据源可能需要（如东方财富、新浪财经等），但AkShare已做封装，用户无需自行申请

### 调用方式
```python
import akshare as ak

# 获取A股历史行情数据
stock_df = ak.stock_zh_a_hist(symbol="000001", period="daily", start_date="20180101", end_date="20240101", adjust="qfq")
print(stock_df)

# 获取实时行情
stock_zh_a_spot_df = ak.stock_zh_a_spot()
print(stock_zh_a_spot_df)
```

> 注意：股票代码格式为 `000001`，不需要加交易所后缀，AkShare内部自动处理

### 注意事项
- 数据来源于各公开渠道（东方财富、新浪财经、同花顺等）
- 封装了非常丰富的数据接口，不仅A股，还支持基金、期货、债券、外汇等
- 因为是聚合数据，数据质量参差不齐，需要验证
- 部分接口可能因为目标网站改版而失效，作者会不定期更新维护

### 频率限制
- 受制于目标网站的反爬策略
- 建议调用间隔设置为1-2秒，避免IP被封
- 批量获取数据时分批请求

### 免费额度
- **完全免费**，无额度限制
- 所有接口开放使用
- 支持获取：
  - A股日线/分钟线实时行情
  - 历史行情数据
  - 资金流向、龙虎榜、股吧热度
  - 财务数据、财报数据
  - 基金、期货、债券、港股、美股等数据

## 4. 百度指数免费API

### 概述
百度指数可以反映关键词的搜索热度，对于市场情绪研判很有帮助。百度官方API需要付费，但有几种免费获取方式。

### 申请步骤（免费方式）

#### 方式一：利用第三方开源库（推荐）
1. 不需要申请百度官方API
2. 直接使用 `bdpy` 或 `baidu-index` 等开源库
```bash
pip install bdpy
```

#### 方式二：百度云官方API（需要注册，有免费额度）
1. 访问 [百度云](https://cloud.baidu.com/) 注册账号
2. 进入「管理控制台」→「产品服务」→「百度指数API」（实际是「内容识别-公众趋势分析」）
3. 创建应用，获取 `API Key` 和 `Secret Key`
4. 每个账号有一定免费调用额度

### 获取token/key流程（官方API）
1. 登录百度云控制台
2. 创建「公众趋势分析」应用
3. 在应用详情中获得：
   - **API Key**（客户端密钥）
   - **Secret Key**（服务器端密钥）
4. 通过这两个key获取access_token：
```python
import requests
url = "https://aip.baidubce.com/oauth/2.0/token"
params = {
    "grant_type": "client_credentials",
    "client_id": "你的API Key",
    "client_secret": "你的Secret Key"
}
response = requests.get(url, params=params)
token = response.json()['access_token']
```

### 调用方式

#### 第三方开源方式（免费无key）：
```python
import bdpy
index = bdpy.BaiduIndex('cookie获取的st')
data = index.get_index(['股票', '牛市'], '2023-01-01', '2024-01-01')
```
> 需要从浏览器登录百度指数后复制cookie中的 `ST` 参数，属于非官方方式，可能失效。

#### 官方API方式：
```python
import requests

url = f"https://aip.baidubce.com/api/v1/feed/word_trend_by_trend?access_token={token}"
payload = {
    "words": ["股票", "基金"],
    "startDate": "2023-01-01",
    "endDate": "2024-01-01",
    "period": "daily"
}
response = requests.post(url, json=payload)
print(response.json())
```

### 注意事项
- **非官方免费方式**：依赖模拟登录，百度可能更新反爬策略，随时可能失效
- **官方API方式**：稳定可靠，但有免费额度限制，超额需要付费
- 获取cookie方式需要手动操作，不适合自动化
- 百度指数只能获得相对指数（0-100），不是绝对搜索量

### 频率限制
- 非官方方式：无明确限制，但IP被封概率高，建议低频调用
- 官方API：**免费用户 QPS=1**，即每秒1次

### 免费额度
- 第三方非官方方式：**完全免费**，但不稳定
- 百度云官方API：
  - 免费配额：**500次调用/月**
  - 超出后：约0.01-0.1元/次，按量付费

## 5. 交易所公开数据

### 概述
上海证券交易所、深圳证券交易所、北京证券交易所官网都提供公开数据下载，完全免费权威。

### 申请步骤
1. **无需注册申请**，完全公开免费
2. 直接访问交易所官网下载板块

#### 上海证券交易所
- 官网：https://www.sse.com.cn/
- 数据入口：`披露/数据` → `行情与交易` → `历史行情`

#### 深圳证券交易所
- 官网：https://www.szse.cn/
- 数据入口：`市场数据` → `行情数据` → `历史行情`

#### 北京证券交易所
- 官网：https://www.bse.cn/
- 数据入口：`信息披露` → `市场数据`

### 获取token/key流程
- **不需要token/key**
- 直接下载文件，或通过公开接口获取

### 调用方式

#### 方式一：手动下载（适合小量数据）
1. 访问交易所官网
2. 选择日期，下载CSV/Excel文件
3. 本地导入分析

#### 方式二：程序化获取（推荐，开源封装接口）
使用AkShare或Tushare已经封装好的接口获取，本质还是从交易所获取：
```python
# AkShare方式获取上交所股票列表
import akshare as ak
sse_stocks = ak.stock_info_sse()
print(sse_stocks)
```

#### 方式三：直接爬取（需要处理反爬）
```python
import requests
# 上交所交易日历
url = "https://www.sse.com.cn/services/trading/tradingcalendar/"
response = requests.get(url)
# 解析HTML获取数据
```

### 注意事项
- 数据最权威最准确，是最终数据源
- 需要自行整理和清洗，格式不统一
- 历史数据只能逐批次下载，比较麻烦
- 没有封装好的API，程序化获取需要写爬虫处理
- 网站反爬不严格，但不要高频请求

### 频率限制
- 官方没有明确限制，建议每分钟不超过10次请求
- 批量下载建议错开时间

### 免费额度
- **完全免费**，无任何额度限制
- 可获取：
  - 完整的历史日线行情
  - 上市公司公告、财报
  - 交易日历、休市安排
  - 新股发行信息
  - 监管信息

## 申请顺序建议

根据**免费程度、易用性、数据完整性**综合建议，按以下顺序申请：

### 第一步：先安装使用 Baostock + AkShare ✅
- **理由**：完全免费，无需注册，安装就能用
- 适合：新手入门、基础量化研究、历史数据分析
- 先用这两个满足大部分需求，不用折腾注册申请

### 第二步：注册 Tushare Pro ✅
- **理由**：数据质量高，接口稳定，文档完善
- 新用户注册就送100积分，足够基础使用

- 社区活跃，遇到问题容易找到解决方案

### 第三步：交易所公开数据补充 ✅
- **理由**：免费权威，作为数据验证和补充
- 当你需要最准确的官方数据，或者某些数据其他接口没有时，从这里获取

### 第四步：百度指数 ✅
- **理由**：属于辅助数据，不是必须的基础行情数据
- 如果你需要做情绪分析、热点跟踪，再申请
- 先试试第三方免费方式，不够用再考虑官方API

### 最终推荐组合
```
基础行情：Tushare Pro（免费积分够用）
补充数据：AkShare（丰富的另类数据）
历史回测：Baostock（免费完整历史数据）
```
