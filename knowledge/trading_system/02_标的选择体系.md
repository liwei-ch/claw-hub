# 标的选择体系（30%权重）
## 🔍 三维选股标准
### 一、基本面筛选（排除项）
✅ **必须满足的条件：**
1. 最新年报/季报净利润为正，扣非净利润为正
2. 净资产收益率（ROE）≥5%
3. 资产负债率≤70%（金融、地产行业除外）
4. 最近一年无重大违法违规、无被证监会立案调查记录
5. 无ST、*ST、退市风险警示
❌ **直接排除的标的：**
1. 最近三个季度连续亏损
2. 大股东质押比例≥80%
3. 最近一年有财务造假、信息披露违规记录
4. 主营业务不清晰，概念炒作无实际业绩支撑
### 二、技术面筛选（准入项）
✅ **短线标的（3~10天）：**
1. 均线排列：5日均线>10日均线>20日均线，呈多头排列
2. 成交量：最近5个交易日成交量较之前放大≥30%
3. 趋势：股价站在20日均线上方，且20日均线向上
4. 波动率：近20日波动率≥3%，流动性充足
✅ **中线标的（1~3个月）：**
1. 均线排列：20日均线>60日均线>120日均线，中长期多头排列
2. 趋势：股价突破近3个月高点，上升趋势明确
3. 量能：突破时成交量较平时放大≥50%，量价配合
✅ **长线标的（6个月以上）：**
1. 均线排列：60日均线>120日均线>250日均线，年线向上
2. 业绩：近三年净利润复合增长率≥15%
3. 行业：属于国家政策支持的高景气赛道
### 三、资金面筛选（加分项）
✅ **优先选择满足以下条件的标的：**
1. 北向资金最近10个交易日连续增持
2. 龙虎榜显示机构席位净买入
3. 融资余额最近1个月增长≥20%
4. 行业板块资金净流入排名前5
## 📊 股票池维护规则
1. 短线股票池：每周更新一次，保留30~50只标的
2. 中线股票池：每月更新一次，保留10~20只标的
3. 长线股票池：每季度更新一次，保留5~10只标的
4. 每日盘后剔除不符合条件的标的，补充新的符合条件的标的
