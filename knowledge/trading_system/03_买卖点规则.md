# 买卖点规则（25%权重）
## 📥 入场信号
### 一、短线入场信号（3~10天）
✅ **信号1：突破入场**
- 股价突破近10日高点，成交量放大≥30%
- 突破时涨幅在3%~5%之间，不追高超过5%
- 止损设置：入场价下跌5%
- 止盈设置：入场价上涨10%~15%
✅ **信号2：回调入场**
- 股价在上升趋势中回调到20日均线附近
- 回调时成交量缩量，跌幅≤3%
- 止损设置：20日均线下方2%
- 止盈设置：前期高点附近
### 二、中线入场信号（1~3个月）
✅ **信号1：趋势突破**
- 股价突破近3个月震荡平台高点
- 突破时成交量放大≥50%，量价配合
- 止损设置：平台下方3%
- 止盈设置：突破高度的1~1.5倍
✅ **信号2：回踩确认**
- 突破后回踩平台上方获得支撑
- 回踩时缩量，未跌破平台
- 止损设置：平台下方2%
- 止盈设置：同突破信号
### 三、长线入场信号（6个月以上）
✅ **信号：估值底部+业绩拐点**
- 市盈率（PE）处于近3年分位值≤30%
- 季度业绩增速环比提升≥20%，业绩拐点明确
- 止损设置：入场价下跌15%
- 止盈设置：估值回到近3年分位值≥70%
## 📤 离场信号
### 一、止损离场（强制规则）
1. 短线：亏损达到5%，无条件止损
2. 中线：亏损达到8%，无条件止损
3. 长线：亏损达到15%，无条件止损
4. 大盘单日下跌≥2%，所有短线持仓止损离场
### 二、止盈离场
1. 短线：盈利达到10%~15%，分批止盈
2. 中线：盈利达到30%~50%，分批止盈
3. 长线：盈利达到100%以上，分批止盈
4. 达到目标价位后，跌破5日均线全部离场
### 三、被动离场信号
1. 股价跌破20日均线，短线全部离场
2. 股价跌破60日均线，中线全部离场
3. 股价跌破年线，长线全部离场
4. 基本面出现重大利空，无条件离场
