# 仓位管理策略（20%权重）
## 📊 总仓位控制规则
### 按大盘趋势调整总仓位
| 大盘趋势 | 总仓位上限 | 说明 |
|----------|------------|------|
| 上升趋势（20日均线向上，指数站在20日均线上方） | 80%~100% | 可以重仓操作 |
| 震荡趋势（指数在20日均线上下震荡，20日均线走平） | 30%~50% | 半仓以下操作，快进快出 |
| 下跌趋势（20日均线向下，指数站在20日均线下方） | 0%~30% | 轻仓或空仓观望 |
### 按时间节点调整仓位
1. 长假前（春节、国庆等）：总仓位≤30%，规避不确定性风险
2. 年报、季报披露期：总仓位≤50%，规避业绩暴雷风险
3. 重大政策发布期：总仓位≤50%，等待政策明朗
## 📈 单只个股仓位限制
1. 单只短线个股仓位上限：20%
2. 单只中线个股仓位上限：30%
3. 单只长线个股仓位上限：40%
4. 同一行业个股总仓位上限：50%，避免行业系统性风险
## 🔄 仓位动态调整机制
### 加仓规则
1. 盈利加仓：只有在持仓已经盈利的情况下才能加仓
2. 加仓比例：每次加仓金额不超过底仓的50%，采用金字塔加仓法
3. 加仓条件：股价回调到关键支撑位，且获得支撑
### 减仓规则
1. 被动减仓：达到止损线无条件减仓/清仓
2. 主动减仓：达到止盈目标分批减仓
3. 风控减仓：大盘趋势变坏，整体减仓
### 分仓原则
1. 短线持仓数量：3~5只，分散风险
2. 中线持仓数量：2~3只，集中精力
3. 长线持仓数量：1~2只，深度研究
4. 禁止单只个股满仓操作，禁止所有资金集中在同一行业
## 💸 头寸计算方法
单笔头寸风险 = 总资金 × 单笔最大风险比例（1%~2%）
持仓数量 = 单笔头寸风险 /（入场价 - 止损价）
*示例：总资金100万，单笔最大风险1%=1万，入场价10元，止损价9.5元，持仓数量=1万/(10-9.5)=2万股，持仓金额20万，占总资金20%，符合仓位限制*
