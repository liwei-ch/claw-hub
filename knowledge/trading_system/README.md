## 🎯 系统核心指标
- 预期年化收益率：20%~30%
- 最大回撤控制：≤15%
- 交易胜率：≥55%
- 盈亏比：≥2:1
## ⚠️ 使用说明
1. 所有交易必须严格按照本系统规则执行，禁止主观臆断
2. 系统规则调整必须经过严格的复盘和模拟交易验证
3. 交易记录必须完整记录，定期复盘优化
4. 风险控制是第一优先级，任何时候都不能违反风险控制规则
