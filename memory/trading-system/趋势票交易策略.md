# 趋势票交易核心策略
## 核心原则（强制执行）
### 一、趋势判断优先级：大周期决定小周期
```
优先级顺序：月线趋势 > 周线趋势 > 日线趋势 > 分时走势
小周期必须服从大周期，大周期趋势未破坏的前提下，小周期回调都是买点而非卖点
```

### 二、大周期趋势判断标准（月线/周线）
#### 1. 上升趋势确认（同时满足）
✅ **均线结构**：5/10/20/60月均线全部多头排列，向上发散
✅ **走势结构**：高点逐步抬高，低点逐步上移，上升结构完整
✅ **量能配合**：上涨放量，调整缩量，量价结构健康
✅ **指标信号**：MACD在零轴上方运行，无顶背离信号
✅ **支撑有效**：每次调整都在20月均线/20周均线处获得支撑

#### 2. 趋势败坏信号（满足任意一个即警惕）
❌ **均线破位**：有效跌破20月均线（连续3周收盘价在均线下方）
❌ **结构破坏**：跌破前期重要低点，上升趋势线被有效击穿
❌ **量价背离**：价格创新高但量能无法同步放大，出现顶背离
❌ **指标走坏**：MACD高位死叉，跌破零轴，红柱持续缩小

### 三、交易执行策略
#### 1. 买点判断（大周期向上前提下）
✅ **黄金买点1**：大周期上升趋势确认，日线级别调整到20/60日均线支撑位
✅ **黄金买点2**：周线级别调整到0.618黄金分割位，出现止跌信号
✅ **黄金买点3**：日线连续调整后出现放量下影线、阳包阴等探底回升K线形态
✅ **仓位策略**：大周期向上时，小周期调整仓位配置比例：
   - 首次接触支撑位：建仓50%
   - 确认支撑有效：加仓30%
   - 突破短期压力位：加仓剩余20%

#### 2. 卖点判断
❌ **止损信号**：大周期上升趋势确认被破坏，有效跌破核心支撑位
❌ **止盈信号**：达到预期目标位，或短期涨幅过大出现放量滞涨
❌ **减仓信号**：日线级别出现明确的见顶信号（M头、顶背离、天量天价等）

### 四、海光信息案例验证
1. **大周期趋势**：月线多头排列，上升结构完整，无败坏信号 → 长期向上确认
2. **中期趋势**：周线调整到0.618黄金分割位，接近20周均线 → 调整到位
3. **短期趋势**：日线级别调整，属于小周期波动 → 服从大周期，是买入机会
4. **策略执行**：43-45元区间分批建仓，目标价75元，预期收益70%+

### 五、纪律规则
1. 永远不做"逆大周期"的交易，大周期向下的票坚决不碰
2. 大周期向上的票，下跌时敢于加仓，而不是恐慌止损
3. 严格区分"短期波动"和"趋势破坏"，前者是机会，后者是风险
4. 所有交易决策必须先看月线，再看周线，最后看日线，顺序不能颠倒
