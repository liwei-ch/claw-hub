# 交易体系知识库
## 📚 交易铁则
### 1. 趋势交易优先级
- 月线趋势 > 周线趋势 > 日线趋势 > 分时走势，小周期必须服从大周期
- 大周期向上前提下，短期回调都是买点
- 只有大周期趋势确认破坏才止损

### 2. 数据校验铁则
- 所有行情数据必须双数据源交叉验证
- 默认显示除权交易价，前复权价需明确标注
- 价格偏离合理区间±30%自动触发二次核验
