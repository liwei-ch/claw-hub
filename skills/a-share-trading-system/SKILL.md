# A股交易系统 - L总专属
**版本：v1.1.1**
**更新时间：2026-03-16**
**核心原则：风险 > 资金利用率 > 进攻力度 > 速度**
**重要更新：取消固定赛道优先级，改为动态板块异动/热点持续性判断规则；新增板块异动统计要求，避免样本偏差**

---

## 🔒 不可动摇的铁律
### 1. 止盈止损铁律
- 能赚钱走的绝不亏钱走，能小亏走绝不大亏走
- 看不准/心里不舒服立刻割肉
- 单票最大回撤不超过15%
- 操作与情绪完全分离，做错立刻认错绝不扛单

### 2. 仓位硬约束
- 总仓位结构：50%趋势波段 + 50%短线
- 趋势波段：3-5只（最佳3只），单票1-2层仓，同一行业1只，单行业总仓位≤2层
- 短线：1-3只，单票0.5-1层仓，快进快出不格局
- 超过持仓数量上限绝不开新仓，新开仓必须先汰弱留强

### 3. 板块判断规则（取消固定优先级，改为动态判断）
#### 新规则：无固定赛道优先级，以实际板块异动和热点持续性为核心判断标准
1. **板块异动判断标准**：
   - 板块涨幅超2%
   - 板块内≥3只个股涨停
   - 板块上涨家数占比≥70%
   - 同时满足以上3个条件判定为板块异动
2. **热点板块持续性判断标准**：
   - 异动板块第二天继续上涨，且有≥2只个股连板 → 判定为短期热点板块，持续性≥2天
   - 板块连续3天上涨，领涨龙头累计涨幅≥30% → 判定为中期主线板块，持续性≥3天
   - 板块持续性≥3天且有政策/行业基本面催化 → 判定为核心主线板块
3. **操作对应规则**：
   - 异动板块出现可小仓位（0.5层）试错领涨龙头
   - 短期热点板块可将仓位提升至1-2层，参与核心标的
   - 中期主线板块可将仓位提升至3-5层，重点配置
   - 核心主线板块可重仓配置，单板块仓位不超过5层

## 📊 情绪量化判断标准
| 情绪阶段 | 量化判断标准 | 操作策略 |
|----------|--------------|----------|
| 🧊 情绪冰点 | *大部分指标同时满足负向极值*<br>1. 跌停数≥20家<br>2. 下跌家数≥4000家<br>3. 连板高度≤3板<br>4. 赚钱效应<30%<br>5. 高位股批量核按钮 | 轻仓（≤1层）试错，或空仓观望 |
| 🚀 情绪主升 | 1. 涨停数≥80家<br>2. 连板高度≥5板<br>3. 赚钱效应≥60%<br>4. 板块持续性≥3天 | 逐步加仓，最高到满仓 |
| 🌊 情绪退潮 | 1. 跌停数≥15家<br>2. 连板高度快速下降<br>3. 赚钱效应<40%<br>4. 高位股出现A字杀 | 清仓短线仓位，趋势票减仓 |
| 🌀 混沌行情 | 1. 涨跌停数相当（30-50家）<br>2. 板块轮动速度≤2天<br>3. 没有明确主线板块<br>4. 成交量持续萎缩 | 最优解：空仓<br>次优解：≤2层仓快进快出 |

## 🎯 选股逻辑
### 龙头原则
- 优先选异动板块/热点板块的领涨龙头/细分龙头，坚决不做杂毛
- 优先选择：上升趋势、机构活跃、资金流入（包括暗盘资金）
- 确定性信号：低位、N字趋势、低位首板、技术+题材+消息+资金共振
- 成交量判断规则：成交量必须和近5日/10日平均量能对比，按量比判断放量/缩量，禁止仅用单日绝对值判断：
  - 量比≥1.5 → 放量
  - 量比≤0.7 → 缩量
  - 0.7<量比<1.5 → 平量

### 板块主线判断
#### 异动判断规则（严格执行，避免样本偏差，多维度综合判断）：
1. 异动判断核心统计维度（统计≥20只板块核心标的，覆盖全产业链）：
   | 维度 | 异动合格标准 |
   |------|--------------|
   | 平均涨幅 | 板块平均涨幅≥2% |
   | 涨停数量 | 板块内≥3只个股涨停（分板块标准：主板≥9%、科创板/创业板≥18%、北交所≥27%，达到涨跌幅限制90%即视为涨停） |
   | 上涨占比 | 板块上涨家数占比≥70% |
   | 大阳数量（涨幅≥5%） | 板块内≥10只个股涨幅≥5% |
   | 大阴数量（跌幅≥5%） | 板块内大阴个股数量≤3只（无明显恐慌抛盘） |
   | 跌停数量 | 板块内跌停个股≤1只 |
   | 资金流入 | 板块整体主力资金净流入≥5亿元（无大额资金出逃） |
   
   ✅ **异动判定标准**：同时满足「平均涨幅≥2% + 涨停≥3只 + 上涨占比≥70%」三个核心条件，且大阴/跌停/资金流入符合要求，即可判定为异动板块。
2. 热点等级判定：
   - 一级热点（超强）：同时满足以上所有6个维度标准，板块持续性≥3天
   - 二级热点（较强）：满足3个核心条件+大阳≥5只，持续性≥2天
   - 三级热点（普通）：仅满足3个核心条件，持续性≥1天
3. 统计要求：
   - 统计板块个股时必须覆盖≥20只核心标的，包含产业链上下游全范围，禁止仅用少量样本股统计
   - 优先采用「全市场涨停个股按板块归类」的方式统计，避免样本遗漏
   - 统计完成后必须复核：交叉验证2个以上数据源/统计方式，确保数据准确
3. 持续性标准：
   - 异动板块次日继续上涨且有≥2只个股连板，判定为短期热点板块
   - 热点板块持续性≥3天且领涨龙头累计涨幅≥30%，判定为中期主线板块
4. 主线标准：昨日成交额前5的板块，今日有4只继续走强就是核心主线板块

## ⏱️ 买卖时机
### 前置原则：大盘情绪优先，情绪不好坚决不操作
#### 买入条件（需同时满足）
1. 板块效应显现，板块龙头领涨
2. 个股止跌企稳，弱转强拐点明确
3. 均线/平台支撑位出现小四连阳、四天三阳、反包K线
4. 有明确的仓位空位（趋势票≤5只）

#### 卖出信号（满足任意一个）
1. 指标转弱（量能萎缩、资金流出、机构活跃度下降）
2. 尾盘跌破支撑位（5/10/20均线）
3. 遇压力位不过
4. 技术形态拐点：倒V、M头附近
5. 放量首阴
6. 趋势破坏，不管盈利亏损立刻走
7. 有确定性更高的新标的，先卖掉走弱的旧仓再换

## 🎚️ 仓位控制规则
### 总原则
向上趋势启动减仓，趋势确认加仓，趋势拐点减仓，趋势破位清仓

### 减仓触发条件（至少减1/2仓位）
- 市场情绪不好：情绪退潮、情绪极度高潮
- 跌破支撑位，破位幅度越大减仓越多
- 黑天鹅/灰犀牛事件
- 情绪极度亢奋、自己心绪不安时
- 单次大亏后，市场进入hard模式
- 持仓数量超过5只时，必须汰弱留强减到上限以内

### 加仓触发条件
- 市场情绪良好，按确定性和信心度逐步加仓
- 弱转强确认：加1-2层
- 资金大幅进场：根据力度加1-2层
- 趋势向上，回踩5日线支撑反弹：加1-2层

## 🌪️ 特殊行情应对
### 混沌行情（指数高位震荡、板块快速轮动）
应对优先级从高到低：
1. 最优解：空仓等待
2. 次优解：降低仓位，小仓位快进快出博弈小修复
3. 替代方案：埋伏主线板块，不追高轮动热点
4. 重仓应对：大趋势标的坚定持有，趋势不破不走

### 极端行情应对
1. 大盘单日跌幅≥3%/千股跌停：无条件清仓所有短线仓位
2. 趋势票系统性破位：先减仓50%，观察企稳信号后再决定去留
3. 黑天鹅事件影响持仓标的：第一时间止损，不抱有侥幸心理
