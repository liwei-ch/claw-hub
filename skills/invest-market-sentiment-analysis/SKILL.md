# 大盘情绪分析 Skill

## 核心功能
1. **多指标自动采集**：
   - 指数趋势：上证指数、深证成指、创业板指的涨跌幅、成交量、均线趋势
   - 市场热度：上涨/下跌/平盘家数、涨停/跌停家数、炸板数/炸板率
   - 资金指标：北向资金净流入/流出、南向资金、两融余额变化
   - 赚钱效应：涨跌中位数、涨幅大于5%/跌幅大于5%个股数量

2. **情绪阶段判定**：
   基于多指标加权评分，将大盘情绪分为5个阶段：
   - 极度恐慌（评分 < 20）
   - 恐慌（20 ≤ 评分 < 40）
   - 中性（40 ≤ 评分 < 60）
   - 贪婪（60 ≤ 评分 < 80）
   - 极度贪婪（评分 ≥ 80）

3. **双源数据校验**：
   同时从东方财富、同花顺两个数据源获取数据，交叉校验确保数据准确性，误差超过5%时自动告警。

4. **Markdown报告输出**：
   自动生成结构化分析报告，包含核心指标、情绪评分、阶段判定、操作建议。

## 使用方法
调用scripts/market_sentiment_analysis.py脚本获取最新情绪分析报告：
```bash
python3 /root/.openclaw/workspace/skills/market-sentiment-analysis/scripts/market_sentiment_analysis.py
```
