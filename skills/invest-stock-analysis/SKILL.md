# 股票个股分析技能（SKILL.md）
## 技能说明
本技能用于对A股个股进行全面专业分析，涵盖基本面、技术面、估值、风险等多维度，输出标准化分析报告。

## 触发条件
用户提问包含以下关键词时自动调用本技能：
- "分析XX股票"、"XX怎么样"、"XX行情"
- "XX估值"、"XX基本面"、"XX技术面"
- 个股名称、股票代码相关提问

## 分析框架（7维度标准模型）
### 1. 公司基本概况
- 核心业务及产品结构
- 行业地位与市场占有率
- 主营业务收入构成及占比
- 核心客户与供应商情况

### 2. 最新市场表现
- 最新股价、涨跌幅、成交量、成交额
- 市值（总市值/流通市值）、换手率
- 估值指标（市盈率TTM、市净率、市销率）
- 阶段性涨跌幅（近1周/1月/3月/年初至今/近1年）
- 与行业平均涨跌幅对比

### 3. 技术面分析（强制执行）
#### 3.1 大周期趋势
- 月线/周线K线走势，长期趋势判断（上升/下降/震荡）
- 长期关键压力位与支撑位

#### 3.2 短期趋势
- 日K线走势分析
- 5/10/20/30日均线排列形态（多头/空头/粘合）
- 短期压力位与支撑位（精确到具体价位）

#### 3.3 K线形态识别
- 整理形态：平台、箱体、三角形、旗形等
- 反转形态：M头、W底、V型反转、倒V型、U型底、头肩顶/底等
- 形态完成度与有效性判断

#### 3.4 量价配合分析
- 成交量与价格走势匹配性（价涨量增/价跌量缩等）
- 量能异常变化分析（放量/缩量的市场含义）

### 4. 财务业绩分析
- 最近3年核心财务指标（营收、归母净利润、扣非净利润、毛利率、净利率、ROE）
- 最新季度业绩同比/环比变动情况
- 盈利能力、偿债能力、运营能力分析
- 业绩增长驱动因素分析

### 5. 核心竞争优势
- 技术壁垒、研发投入与专利情况
- 渠道与品牌优势
- 成本控制能力
- 政策受益情况
- 业务拓展与新产品布局

### 6. 风险与挑战
- 行业竞争格局与竞争加剧风险
- 政策变动风险（行业监管、贸易政策等）
- 上游原材料价格波动风险
- 下游需求不及预期风险
- 公司自身经营风险（股权质押、商誉减值等）

### 7. 投资价值分析
- 估值水平与行业平均、历史分位对比
- 未来增长看点与业绩驱动因素
- 投资评级（推荐/谨慎推荐/中性/回避）
- 操作建议（支撑位、压力位、适合的操作策略）
- 投资周期建议（短期/中期/长期）

## 数据质量控制规则（强制执行）
### 1. 数据源要求
- 行情数据：至少交叉验证2个独立数据源（东方财富/同花顺/新浪财经/雪球任选其二）
- 财务数据：优先使用公司官方公告、交易所披露信息
- 行业数据：使用行业协会、国家统计局等官方机构发布数据

### 2. 数据校验规则
- 百分比类数据必须手动核算：`(当期值-基期值)/基期值×100%`
- 异常值识别：数据偏离行业平均±30%时必须二次核实
- 时间维度校验：不同周期数据逻辑自洽

| 3 | 当前价 | 数值 |
| 4 | 昨收价 | 数值 |

## 报告输出规范
1. 结构清晰，使用层级标题和分点表述
2. 关键数据使用表格展示，便于阅读
3. 所有结论需有数据支撑，避免主观臆断
4. 末尾必须包含标准化风险提示：
   > **风险提示**：以上分析基于公开信息整理，不构成投资建议，股市有风险，投资需谨慎。
5. 涉及具体操作建议时标注："仅供参考，不构成买卖依据"

## 质量控制流程
```
信息收集 → 多数据源交叉验证 → 数据核算校验 → 分析撰写 → 逻辑合理性检查 → 报告输出
```
