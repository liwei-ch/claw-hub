# 趋势票分析工具

## 核心功能
输入股票代码，自动获取数据，进行三级趋势分析（月线→周线→日线），判断趋势方向，给出买卖建议，输出markdown格式分析报告。

## 使用方法

### 基本使用
```bash
python ~/.openclaw/workspace/skills/trend-stock-analysis/scripts/trend_analysis.py <股票代码>
```
股票代码支持两种格式：纯数字（如000001）或带后缀（如000001.SZ）

### 数据来源
本工具遵循严格的数据双源校验规则：
1. 主要数据源：Tushare Pro（需要配置TUSHARE_TOKEN环境变量）
2. 备用数据源：akshare（免费开源财经数据接口）
3. 双源校验：两个数据源获取的数据差异在合理范围内才输出结论

## 分析逻辑

### 三级趋势判断顺序（月线→周线→日线）
遵循"看大做小"原则：
1. **月线**：判断长期趋势方向（上升/下降/震荡）
2. **周线**：判断中期趋势方向，确认月线趋势
3. **日线**：判断短期趋势方向，寻找买卖点

### 趋势判断标准
| 趋势类型 | 判断标准 |
|----------|----------|
| 上升趋势 | 均线多头排列（短期均线在长期均线之上）+ 股价创新高 + 高点逐步抬高 |
| 下降趋势 | 均线空头排列（短期均线在长期均线之下）+ 股价创新低 + 低点逐步降低 |
| 震荡趋势 | 均线交织 + 股价在区间内波动 |

### 买卖建议规则
| 趋势组合（月/周/日） | 建议操作 |
|----------------------|----------|
| 上升/上升/上升 | 买入/持有，回踩均线加仓 |
| 上升/上升/回调 | 持有，等待回调结束加仓 |
| 上升/下降/任何 | 观望，不操作 |
| 下降/任何/任何 | 坚决不买，持有的止损出局 |
| 震荡/上升/上升 | 轻仓试错，突破加仓 |
| 震荡/任何/下降 | 观望 |

## 输出格式
自动生成markdown格式分析报告，包含：
1. 股票基本信息（代码、名称、所属行业）
2. 月线趋势分析
3. 周线趋势分析
4. 日线趋势分析
5. 综合判断
6. 买卖建议
7. 风险提示
