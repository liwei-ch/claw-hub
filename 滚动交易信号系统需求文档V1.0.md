# 滚动交易信号自动分析系统 需求文档V1.0
## 📋 需求背景
当前针对深南电路等中长期趋势标的的日内滚动交易，需要每日人工分析大盘、板块、个股信号进行操作决策，效率较低且存在主观判断偏差，需要开发自动化信号研判系统，实现信号自动分析、策略自动生成、结果自动推送，降低人工成本，提高操作胜率。
## 🎯 核心目标
1. 每日**9:25集合竞价结束后5分钟内**自动输出当日滚动交易策略
2. 信号研判准确率≥80%
3. 完全符合我们的交易体系规则，无主观判断偏差
## 🔧 功能模块设计
### 1. 数据采集模块（每日9:25自动触发）
| 数据类型 | 采集指标 | 数据源 | 更新频率 | 优先级 |
|----------|----------|--------|----------|--------|
| 大盘数据 | 上证指数涨跌幅、创业板指涨跌幅、全市场上涨家数/下跌家数、赚钱效应（上涨家数/总家数）、涨跌停家数 | 东方财富公开API、同花顺免费接口 | 实时 | P0 |
| 板块数据 | 半导体板块涨跌幅、AI算力板块涨跌幅、板块主力资金净流入/流出金额、板块领涨/领跌个股表现 | 东方财富公开API、财联社公开接口 | 实时 | P0 |
| 持仓个股数据 | 深南电路(002916)开盘价、量比、涨跌幅、竞价量能 | 东方财富公开API | 实时 | P0 |
| 消息面数据 | 半导体/AI行业政策、个股重大公告、行业突发消息 | 财联社快讯公开接口 | T+1分钟 | P1 |
### 2. 信号研判引擎（内置规则不可修改）
#### 2.1 信号打分规则
| 判断维度 | 正向信号（+1分） | 反向信号（-1分） |
|----------|------------------|------------------|
| 大盘环境 | 上证指数红盘/创业板红盘，赚钱效应≥60% | 上证指数绿盘/创业板绿盘，赚钱效应≤40% |
| 板块状态 | 半导体/AI板块涨跌幅≥1%，主力资金净流入≥1亿 | 半导体/AI板块涨跌幅≤-1%，主力资金净流出≥1亿 |
| 个股表现 | 个股高开/低开高走，量比≥1.5（放量） | 个股低开/高开低走，量比≤0.7（缩量） |
| 消息面 | 行业有正面政策/利好消息 | 行业有负面政策/利空消息 |
#### 2.2 最终信号判定规则
| 总分区间 | 操作类型 | 说明 |
|----------|----------|------|
| ≥2分 | ✅ 正向滚动（优先推荐） | 信号明确，优先执行正向滚动 |
| ≤-2分 | ⚠️ 反向滚动（谨慎操作） | 信号明确，可执行反向滚动 |
| -1分 ~ 1分 | ❌ 观望 | 信号不明确，放弃当日滚动操作 |
### 3. 策略生成模块
根据最终信号自动输出可直接执行的操作策略：
| 操作类型 | 策略输出内容（固定格式） |
|----------|--------------------------|
| 正向滚动 | 1. 滚动仓位：底仓的50%（单次不超过总资金1/6）<br>2. 买点区间：244-245元（下跌1%-2%有承接）<br>3. 卖点区间：249-250元（涨幅2%以上滞涨）<br>4. 止损阈值：滚动仓亏损1%立即平仓 |
| 反向滚动 | 1. 滚动仓位：底仓的50%（单次不超过总资金1/6）<br>2. 卖点区间：252元以上（涨幅2%+滞涨）<br>3. 买点区间：247元以下（下跌有承接）<br>4. 止损阈值：滚动仓亏损1%立即平仓 |
| 观望 | 当日无明确信号，建议放弃滚动操作，持有底仓即可 |
### 4. 消息推送模块
- 推送时间：每日9:30前
- 推送渠道：当前飞书群（oc_9aa475cf3c0aeeb88c3e33e0b7a8b3b8）
- 推送格式：Markdown卡片，清晰易读，关键信息加粗高亮
## 📌 技术要求
1. **数据源要求**：全部使用公开免费API，无需付费接口，优先选用稳定性高、延迟低的公开数据源，备用数据源≥2个，主数据源故障自动切换
2. **稳定性要求**：每日9:25必须触发采集分析，连续运行可用性≥99.9%，异常情况自动报警
3. **规则可配置性**：打分规则、价格区间、仓位比例支持后台配置修改，无需重新部署代码
4. **数据留存要求**：所有历史信号、策略、操作结果自动存入数据库，支持后续复盘和规则优化

```markdown
📅 2026年3月17日 深南电路滚动交易策略
🔍 信号研判得分：+2分（正向信号明确）

- 消息面：无重大消息（+0）
---
💡 操作指令：
1. 滚动仓位：底仓的50%（总资金1/6）
2. 买点：244-245元区间，跌不动有承接时买入
3. 卖点：249-250元区间，滞涨时卖出
4. 止损：滚动仓亏损1%立即平仓
```
## ✅ 验收标准
1. 功能验收：连续3个交易日正常输出策略，推送时间在9:30前
2. 准确率验收：连续5个交易日信号研判准确率≥80%，与人工判断一致率≥90%
3. 稳定性验收：连续运行7天无异常故障，数据源切换正常
